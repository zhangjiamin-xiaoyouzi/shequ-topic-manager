# 社区话题管理系统

 社区话题管理模块，支持话题列表管理、新建/编辑话题、话题状态筛选等功能。

## 功能

- **话题列表** - 话题搜索、标签筛选、状态筛选、分页浏览
- **新建话题** - 基础样式、业务属性、智能时效、分发位置四大模块配置
- **编辑话题** - 支持草稿/已发布话题的编辑
- **状态管理** - 草稿、已发布、已下架、待审核等状态流转

## 技术栈

- 纯 HTML + CSS + JavaScript
- 设计规范 (Meiyou Design)
-  localStorage 数据持久化
-  postMessage 跨页面通信
